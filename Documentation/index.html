Doctris - React Js Doctor Appointment Booking System

Introduction

Doctris is a clean, modern, creative & responsive HTML template and Admin Dashboard for appointment booking system. It is built with React 18.x & Bootstrap v5.x. This can be used for any health-related website such as doctors clinic, hospital, doctor consultation center, cosmetic surgery center, medical laboratory, dental clinic, centenary clinic, pharmacy, etc. Fully responsive design layouts with visitor-centric stunning look and feel will help reach the business's target audience.

Dark & Light version Available: We have added a Dark & Light Theme

How to change the fonts family in Doctris?

If you want to change your font family then Please first of all open the _fonts.scss (path: src/app/assets/scss/custom/fonts/_fonts.scss) and add your font family @import url('https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap'); Import the @import link from your Google Fonts.

And also add or change to your font family name in _variables.scss & _variables-dark.scss (path: src/app/assets/scss/_variables.scss) and also Dark mode for (path: src/app/assets/scss/_variables-dark.scss),
$font-family-sans-serif:      'Inter', sans-serif;
$font-family-secondary:      'Inter', sans-serif;

How to change the Template or Theme Mode (Light or Dark) in Doctris?

If you change the theme or template mode Light to Dark please follow the below step, Please first of all open the App.js (path: src/App.js) find the import './assets/scss/style.scss' & import './assets/scss/bootstrap.scss' and replace or changes with import './assets/scss/style-dark.scss' & import './assets/scss/bootstrap-dark.scss'

Note:
Light Mode:      import './assets/scss/style.scss' & import './assets/scss/bootstrap.scss'
Light Mode:      import './assets/scss/style-dark.scss' & import './assets/scss/bootstrap-dark.scss'

How to change the Theme primary color in Doctris?

If you change the theme or template color please follow the below step, Please first of all open the _variables.scss & _variables-dark.scss (path: src/app/assets/scss/_variables.scss) and also Dark mode for (path: src/app/assets/scss/_variables-dark.scss) find the $blue and replace or changes with color code

like:
$blue:      #396cf0; replace only #396cf0 to your color (#f89d36)
